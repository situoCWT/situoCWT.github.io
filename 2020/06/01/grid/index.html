2020-06-01 • 标签: css


  网格布局（Grid）是最强大的 CSS 布局方案。 它将网页划分成一个个网格，可以任意组合不同的网格，做出各种各样的布局。以前，只能通过复杂的 CSS 框架达到的效果，现在浏览器内置了。

一.基本概念

二.行、列和单元格


  在容器内，由子元素水平排列而成的区域，我们称之为”行”(Row)，由子元素垂直排列而成的区域，我们称之为”列”(Column)，”行”和”列”交叉的区域，称之为”单元格”(cell)。
