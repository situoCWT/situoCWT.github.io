grid布局

2020-06-01 • 标签: css


  网格布局（Grid）是最强大的 CSS 布局方案。 它将网页划分成一个个网格，可以任意组合不同的网格，做出各种各样的布局。以前，只能通过复杂的 CSS 框架达到的效果，现在浏览器内置了。

二.行、列和单元格


  在容器内，水平排列而成的区域，我们称之为”行”(Row)，垂直排列而成的区域，我们称之为”列”(Column)，”行”和”列”交叉的区域，称之为”单元格”(cell)。

 3.单元格内容的位置


  grid除了可以设置单元格的大小，还可以给单元格内的子元素设置位置。

  justify-items 属性,可以设置子元素在单元格内的水平位置，align-items 属性,可以设置子元素在单元格内的垂直位置,两个属性的取值是一样的分别有以下4种。

  1. start
  2. end
  3. center
  4. stretch(默认值)

  当设置了justify-items: start属性时，单元格的内容会对齐单元格的起始边缘，即左对齐。

  同理，当设置justify-items: end属性时，单元格的内容会对齐单元格的结束边缘，即右对齐。

  justify-items: center居中对齐

  stretch是默认值，代表拉伸，占满整个单元格宽度(当子元素未定义大小时)

  align-items的作用和justify-items的效果一样，只是一个作用于水平方向，一个作用于垂直方向，就不一一演示了，如果要同时设置这两个属性时，可以使用place-items这个属性，这个属性传入两个值，
第一个是代表align-items的值，第二个是代表justify-items的值，当只传入一个值时，第二个值默认与第一个值相同，写法为place-items: <align-items> <justify-items>，例如place-items: center end,代表在水平方向右对齐，垂直方向居中。

 3.内容区域在容器的位置


  当未使用浮动大小时(若使用浮动宽度或者未设置大小，内容区域会占满整个容器)，我们可以对内容区域在容器内的位置进行设置。
  justify-content 属性,可以设置内容区域在容器内的水平位置，align-content 属性,可以设置内容区域在容器内的垂直位置,两个属性的取值有以下几种。

  1. start
  2. end
  3. center
  4. stretch
  5. space-around
  6. space-between
  7. space-evenly

  以水平方向justify-content为例，此时设置值为start,则内容区域会在容器内部左边

  值为end时,则内容区域会在容器内部右边

  值为center时,则内容区域会在容器内部居中

  值为stretch时,当内容区域没有指定大小时，会拉伸占据整个容器

  值为space-around时,每个项目的间隔相等，但是项目与容器边框的间隔是项目与项目之间的一半

  值为space-between时,每个项目的间隔相等，但是项目与容器边框没有间隔

  值为space-evenly时,每个项目两侧的间隔相等，且项目与容器边框的间隔与项目和项目之间的间隔一样大

四.结语


  grid布局可以非常方便的组合出各式各样的布局，基本能满足我们日常开发中的布局效果，而且grid布局的功能还远远不止我上面所写的内容，有兴趣的小伙伴可以更加深入的去了解grid布局，相信一定会有很大的收获的。
